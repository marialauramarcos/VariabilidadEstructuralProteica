Estudio variabilidad estructural proteica

Conjuntos de proteinas

Conjunto teorico

170 mutantes de 1jiaa (122 sitios) simuladas con el modelo mutacional LFENM con ANM R0 = 8 y Fmax = 2 (solo para LFENM).

Con seleccion natural (ns = T): para determinar que sitios mutar, se alineo 1jiaa con cada una de las proteinas del conjunto, se obtuvieron los indices de los sitios alineados pero mutados y de los sitios con gaps y se simularon 10 mutantes por cada par de proteinas, obteniendose un total de 170 mutantes. Luego, para el analisis, se tomaron como alineados los sitios alineados de 1jiaa para cada par de proteinas.

Sin seleccion natural (ns = F): se mutaron al azar la cantidad de sitios correspondientes al % id del conjunto experimental y luego, para el analisis, se tomaron como alineados todos los sitios.

Conjunto experimental

17 proteinas de la familia de phoslip. Las proteinas tienen un promedio de identidad secuencial de 54%. El core del alineamiento de la familia es de 111 sitios.

Analisis

Se realizaron 2 tipos de analisis:

Core (C = T): se analizo solo el core conservado del alinemiento (sitios sin gaps en el alineamiento multiple).

No Core (C = F): no se utilizo el core para el analisis.
