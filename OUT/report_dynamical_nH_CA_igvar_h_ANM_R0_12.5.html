Introduction

We study the evolutionary divergence of proteins using the LF-ENM and comparing simulated mutants with experimental data.

We studied the igvar_h family. We used the ENM ANM with R0 = 12.5 to generate mutants. We used as reference for the ENM the protein with pdb code 1fdlhv. We simulated 10 mutants of this reference for each protein of the family. The igvar_h family is of the type all beta and consists of the following proteins:
