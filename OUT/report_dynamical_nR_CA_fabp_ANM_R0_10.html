Introduction

We study the evolutionary divergence of proteins using the LF-ENM and comparing simulated mutants with experimental data.

We studied the fabp family. We used the ENM ANM with R0 = 10 to generate mutants. We used as reference for the ENM the protein with pdb code 1hmt. We simulated 10 mutants of this reference for each protein of the family. The fabp family is of the type beta and consists of the following proteins:
